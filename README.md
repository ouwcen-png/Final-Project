# Migration Explorer

Migration Explorer is a lightweight interactive prototype for INFOSCI 301. It helps users explore and compare global migration corridors through a simple dashboard with a world overview panel, ranked corridor comparison, filters, and a details panel.

## Repository Structure

- `src/` interface files
- `data/` sample data and metadata
- `assets/` figures for the report
- `docs/` report files

## How to Run

1. Download the repository.
2. Open `src/index.html` in a browser.
3. Use the theme filter to change the visible corridors.
4. Hover over a ranked bar to see detailed information.

## Prototype Features

- corridor comparison
- simple filtering
- hover-based details
- transparent use of aggregated public data
- uncertainty-aware interpretation

## Data Sources

Primary prototype references:
- UN DESA migration data structure
- World Bank contextual indicator structure

This repo currently includes a small sample dataset for interim implementation and demonstration purposes.

## Accessibility

The interface uses:
- high contrast text
- readable labels
- simple layout
- minimal cognitive overload
- visible interaction cues
